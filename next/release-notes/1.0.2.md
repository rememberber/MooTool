# MooTool Next Electron 1.0.2

## English

> Product: MooTool Next Electron<br>
> Version: 1.0.2<br>
> This release is installed and updated independently from MooTool Java and does not overwrite the Java edition.

### Features

- Added code highlighting, brace matching, code folding, and formatting for Notes, JSON Workbench, and other suitable text editing areas.
- Enhanced Find & Replace in Notes and JSON Workbench: auto-fill the search box from the current selection, and support regular expressions and other Java-parity capabilities.

### Fixes and improvements

- Fixed window activation focus so it returns to the ribbon content (the Notes / JSON Workbench editor) instead of the navigation Settings button.
- Improved Translation performance so it is no longer noticeably slower than the Java edition.

### Notes

- Installers remain unsigned. On macOS, the app downloads and verifies the new DMG, then asks the user to open it and drag the app into Applications.
- The Java product line continues to use `v*` tags and its existing update manifest.

## 中文

> 产品线：MooTool Next Electron<br>
> 版本：1.0.2<br>
> 本版本与 MooTool Java 独立安装、独立更新，不会覆盖 Java 版本。

### 新功能

- 随手记、JSON 工作台及其他适用的文本编辑区域支持代码高亮、括号匹配、代码折叠与格式化。
- 增强随手记与 JSON 工作台的查找替换：选中文本时可自动填入搜索框，并支持正则表达式等与 Java 版对齐的能力。

### 修复与改进

- 修复窗口激活时焦点落在导航栏设置按钮的问题；现在会回到功能区内容（随手记 / JSON 工作台的文本编辑框）。
- 优化翻译功能性能，不再明显慢于 Java 版。

### 说明

- 安装包仍未签名。macOS 会下载并校验新版 DMG，随后提示用户打开 DMG 并将应用拖入“应用程序”。
- Java 产品线继续使用 `v*` tag 和原有更新清单，不受本次发布影响。

## 日本語

> 製品ライン：MooTool Next Electron<br>
> バージョン：1.0.2<br>
> このリリースは MooTool Java とは独立してインストール・更新され、Java 版を上書きしません。

### 新機能

- クイックノート、JSON ワークベンチなど対応するテキスト編集領域で、コードハイライト、括弧マッチング、コード折りたたみ、フォーマットに対応しました。
- クイックノートと JSON ワークベンチの検索／置換を強化：選択中のテキストを検索欄へ自動入力し、正規表現など Java 版と同等の機能をサポートします。

### 修正と改善

- ウインドウがアクティブになったときのフォーカスがナビゲーションの設定ボタンに当たる問題を修正し、リボン内の内容（クイックノート／JSON ワークベンチのテキスト編集欄）へ戻るようにしました。
- 翻訳機能のパフォーマンスを改善し、Java 版より明らかに遅い状態を解消しました。

### 補足

- インストーラーは引き続き未署名です。macOS では新しい DMG をダウンロードして検証し、その後 DMG を開いてアプリを「アプリケーション」へドラッグするよう案内します。
- Java 製品ラインは引き続き `v*` タグと従来の更新マニフェストを使用し、このリリースの影響を受けません。
