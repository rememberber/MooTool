# MooTool Next Electron 1.0.7

## English

> Product: MooTool Next Electron<br>
> Version: 1.0.7<br>
> This release is installed and updated independently from MooTool Java and does not overwrite the Java edition.

### Improvements

- Optimized layouts across the Calculator, Workbench, and shared desktop styles for clearer grouping and spacing.
- Redesigned the Calculator into a clearer two-column layout with dedicated arithmetic, base conversion, and number-theory sections.

### Fixes

- Fixed the remaining issue where selecting text in the Quick Notes sidebar preview lost focus and prevented proper text selection.

### Notes

- Installers remain unsigned. On macOS, the app downloads and verifies the new DMG, then asks the user to open it and drag the app into Applications.
- The Java product line continues to use `v*` tags and its existing update manifest.

## 中文

> 产品线：MooTool Next Electron<br>
> 版本：1.0.7<br>
> 本版本与 MooTool Java 独立安装、独立更新，不会覆盖 Java 版本。

### 改进

- 优化计算器、工作区及全局桌面样式布局，分组与间距更清晰。
- 重新设计计算器为更清晰的两栏布局，分别呈现四则运算、进制转换与数论计算区域。

### 修复

- 修复随手记侧边栏预览区选中文本时焦点丢失、无法正常选择的问题（1.0.6 中仍存在的问题）。

### 说明

- 安装包仍未签名。macOS 会下载并校验新版 DMG，随后提示用户打开 DMG 并将应用拖入“应用程序”。
- Java 产品线继续使用 `v*` tag 和原有更新清单，不受本次发布影响。

## 日本語

> 製品ライン：MooTool Next Electron<br>
> バージョン：1.0.7<br>
> このリリースは MooTool Java とは独立してインストール・更新され、Java 版を上書きしません。

### 改善

- 電卓、ワークベンチ、共通デスクトップスタイルのレイアウトを最適化し、グループ分けと余白をより明確にしました。
- 電卓を 2 カラム構成に再設計し、四則演算・基数変換・数論計算の各セクションを分かりやすく整理しました。

### 修正

- クイックノートのサイドバープレビューでテキスト選択時にフォーカスが失われ、正常に選択できない問題（1.0.6 でも残っていた問題）を修正しました。

### 補足

- インストーラーは引き続き未署名です。macOS では新しい DMG をダウンロードして検証し、その後 DMG を開いてアプリを「アプリケーション」へドラッグするよう案内します。
- Java 製品ラインは引き続き `v*` タグと従来の更新マニフェストを使用し、このリリースの影響を受けません。
