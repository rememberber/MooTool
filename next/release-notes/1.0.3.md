# MooTool Next Electron 1.0.3

## English

> Product: MooTool Next Electron<br>
> Version: 1.0.3<br>
> This release is installed and updated independently from MooTool Java and does not overwrite the Java edition.

### Features

- On first launch, show a hint that Java edition data can be migrated from Settings without losing the original Java data.
- Hovering the silent-download ready prompt now shows a friendly summary of the update contents.
- Notes / JSON Workbench formatting now covers more languages such as SQL, and formatting shortcuts match the Java edition.
- Scheduled update checks now run once every hour.

### Notes

- Installers remain unsigned. On macOS, the app downloads and verifies the new DMG, then asks the user to open it and drag the app into Applications.
- The Java product line continues to use `v*` tags and its existing update manifest.

## 中文

> 产品线：MooTool Next Electron<br>
> 版本：1.0.3<br>
> 本版本与 MooTool Java 独立安装、独立更新，不会覆盖 Java 版本。

### 新功能

- 首次安装时提示可从设置迁移 Java 版数据，且不会丢失原有 Java 版数据。
- 静默下载安装包完成后的提示，鼠标悬停可友好展示本次更新内容详情。
- 随手记 / JSON 工作台格式化支持更多语言类型（如 SQL），格式化快捷键与 Java 版保持一致。
- 定时检查更新频率调整为每小时一次。

### 说明

- 安装包仍未签名。macOS 会下载并校验新版 DMG，随后提示用户打开 DMG 并将应用拖入“应用程序”。
- Java 产品线继续使用 `v*` tag 和原有更新清单，不受本次发布影响。

## 日本語

> 製品ライン：MooTool Next Electron<br>
> バージョン：1.0.3<br>
> このリリースは MooTool Java とは独立してインストール・更新され、Java 版を上書きしません。

### 新機能

- 初回起動時に、設定から Java 版データを移行できる案内を表示します（元の Java 版データは失われません）。
- サイレントダウンロード完了の案内にマウスを乗せると、今回の更新内容をわかりやすく表示します。
- クイックノート／JSON ワークベンチのフォーマットが SQL などより多くの言語に対応し、ショートカットも Java 版と揃えました。
- 定期の更新チェック間隔を 1 時間ごとに変更しました。

### 補足

- インストーラーは引き続き未署名です。macOS では新しい DMG をダウンロードして検証し、その後 DMG を開いてアプリを「アプリケーション」へドラッグするよう案内します。
- Java 製品ラインは引き続き `v*` タグと従来の更新マニフェストを使用し、このリリースの影響を受けません。
