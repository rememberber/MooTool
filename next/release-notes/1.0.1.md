# MooTool Next Electron 1.0.1

## English

> Product: MooTool Next Electron<br>
> Version: 1.0.1<br>
> This release is installed and updated independently from MooTool Java and does not overwrite the Java edition.

### Fixes and improvements

- Fully localized the system application menus in Chinese, English, and Japanese.
- Added direct menu access to About, Sync and Backup, Keyboard Shortcuts, Appearance, Layout, Runtimes, and Tool Defaults.
- Fixed **Check for Updates** so it opens the About page and displays the update result.
- Fixed tooltips remaining visible after closing or switching away from a window when focus landed on the Settings button.
- Refined layouts and spacing across the workspace and settings interfaces.

### Notes

- Installers remain unsigned. On macOS, the app downloads and verifies the new DMG, then asks the user to open it and drag the app into Applications.
- The Java product line continues to use `v*` tags and its existing update manifest.

## 中文

> 产品线：MooTool Next Electron<br>
> 版本：1.0.1<br>
> 本版本与 MooTool Java 独立安装、独立更新，不会覆盖 Java 版本。

### 修复与改进

- 完整支持系统应用菜单的中文、英文和日文显示。
- 在系统菜单中增加“关于”“同步与备份”“快捷键”“外观”“布局”“运行环境”和“工具默认值”等快捷入口。
- 修复“检查更新”点击后无反馈的问题；现在会打开“关于”页面并展示检查结果。
- 修复关闭窗口或切换窗口后，焦点落在设置按钮时工具提示仍然残留的问题。
- 优化工作区与设置界面的布局和间距。

### 说明

- 安装包仍未签名。macOS 会下载并校验新版 DMG，随后提示用户打开 DMG 并将应用拖入“应用程序”。
- Java 产品线继续使用 `v*` tag 和原有更新清单，不受本次发布影响。

## 日本語

> 製品ライン：MooTool Next Electron<br>
> バージョン：1.0.1<br>
> このリリースは MooTool Java とは独立してインストール・更新され、Java 版を上書きしません。

### 修正と改善

- システムのアプリケーションメニューを中国語、英語、日本語に完全対応しました。
- 「MooTool について」「同期とバックアップ」「キーボードショートカット」「外観」「レイアウト」「実行環境」「ツールのデフォルト」へのメニュー項目を追加しました。
- 「アップデートを確認」をクリックしても反応しない問題を修正し、「MooTool について」画面に確認結果を表示するようにしました。
- ウインドウを閉じたり切り替えたりした後、設定ボタンにフォーカスが残るとツールチップが表示されたままになる問題を修正しました。
- ワークスペースと設定画面のレイアウトおよび余白を改善しました。

### 補足

- インストーラーは引き続き未署名です。macOS では新しい DMG をダウンロードして検証し、その後 DMG を開いてアプリを「アプリケーション」へドラッグするよう案内します。
- Java 製品ラインは引き続き `v*` タグと従来の更新マニフェストを使用し、このリリースの影響を受けません。
