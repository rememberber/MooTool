# MooTool Next Electron 1.0.6

## English

> Product: MooTool Next Electron<br>
> Version: 1.0.6<br>
> This release is installed and updated independently from MooTool Java and does not overwrite the Java edition.

### Features

- Aligned the system tray menu with the Java edition.
- Added custom-area screenshot capture with a screen overlay and tray-menu shortcuts.
- Added screen color picking: hide the app window, pick a color on screen, then restore the window.
- Quick Notes now supports dragging and inserting images from external sources.

### Fixes

- Fixed the second inserted image not automatically wrapping to a new line.
- Fixed inaccurate focus when selecting text in the Quick Notes sidebar preview.

### Notes

- Installers remain unsigned. On macOS, the app downloads and verifies the new DMG, then asks the user to open it and drag the app into Applications.
- The Java product line continues to use `v*` tags and its existing update manifest.

## 中文

> 产品线：MooTool Next Electron<br>
> 版本：1.0.6<br>
> 本版本与 MooTool Java 独立安装、独立更新，不会覆盖 Java 版本。

### 新功能

- 系统托盘菜单对齐 Java 版实现。
- 截图支持自定义区域：点击后隐藏窗口并显示半透明遮罩，可框选区域后截图；相关快捷操作已加入托盘菜单。
- 屏幕取色：点击后隐藏应用窗口，在屏幕上取色后恢复窗口。
- 随手记支持从外部拖拽并插入图片。

### 修复

- 修复第二张图片插入时未自动换行的问题。
- 修复随手记侧边栏预览区选中文本时焦点不准确、无法正常选择的问题。

### 说明

- 安装包仍未签名。macOS 会下载并校验新版 DMG，随后提示用户打开 DMG 并将应用拖入“应用程序”。
- Java 产品线继续使用 `v*` tag 和原有更新清单，不受本次发布影响。

## 日本語

> 製品ライン：MooTool Next Electron<br>
> バージョン：1.0.6<br>
> このリリースは MooTool Java とは独立してインストール・更新され、Java 版を上書きしません。

### 新機能

- システムトレイメニューを Java 版の実装に合わせました。
- カスタム領域のスクリーンショットに対応：クリック後にウインドウを隠し半透明オーバーレイで範囲選択して撮影できます。関連ショートカットをトレイメニューに追加しました。
- 画面から色を取得：クリック後にアプリを隠して画面上で色を取得し、ウインドウを復帰します。
- クイックノートで外部から画像をドラッグして挿入できます。

### 修正

- 2 枚目の画像挿入時に自動改行されない問題を修正しました。
- クイックノートのサイドバープレビューでテキスト選択時のフォーカスがずれる問題を修正しました。

### 補足

- インストーラーは引き続き未署名です。macOS では新しい DMG をダウンロードして検証し、その後 DMG を開いてアプリを「アプリケーション」へドラッグするよう案内します。
- Java 製品ラインは引き続き `v*` タグと従来の更新マニフェストを使用し、このリリースの影響を受けません。
