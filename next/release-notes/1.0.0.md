# MooTool Next Electron 1.0.0

## English

> Product: MooTool Next Electron<br>
> Version: 1.0.0<br>
> This release is installed and updated independently from MooTool Java and does not overwrite the Java edition.

### First stable release

- Rebuilt the desktop client with Electron, Vite, React, and TypeScript.
- Includes the main developer tools from MooTool Java with a new multi-window workspace experience.
- Provides installers for macOS Apple Silicon, macOS Intel, Windows x64, and Linux x64.
- Uses an independent app ID, product name, user data, and update channel, allowing both editions to be installed together.
- Update checks use only the `next-electron` product manifest and automatically select the package for the current OS and architecture.
- Installers are currently unsigned. On macOS, the app downloads and verifies the new DMG in the background, then asks the user to open it and drag the app into Applications.

### Notes

- This is the initial release of the MooTool Next Electron product line; its versions evolve independently from `1.0.0`.
- The Java product line continues to use `v*` tags and its existing update manifest.

## 中文

> 本版本与 MooTool Java 独立安装、独立更新，不会覆盖 Java 版本。

## 首个正式版本

- 使用 Electron、Vite、React 和 TypeScript 重写桌面客户端。
- 覆盖 MooTool Java 的主要开发者工具，并采用新的多窗口和工作区体验。
- 支持 macOS Apple Silicon、macOS Intel、Windows x64 和 Linux x64 安装包。
- 使用独立应用 ID、产品名称、用户数据和更新通道，可与 MooTool Java 同时安装。
- 更新检查只读取 `next-electron` 产品清单，并自动选择当前系统与架构对应的安装包。
- 本版本安装包暂未进行代码签名；macOS 可在后台下载并校验新版 DMG，完成后会提示打开 DMG，由用户将应用拖入“应用程序”完成安装。

## 更新说明

- 这是 MooTool Next Electron 产品线的起始版本，后续版本号从 `1.0.0` 独立演进。
- Java 产品线继续使用 `v*` tag 和原有更新清单，不受本次发布影响。
